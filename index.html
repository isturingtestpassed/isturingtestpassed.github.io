Is Turing Test Passed?

NO.

What's a Turing Test?

The Turing test is a test of a machine's ability to exhibit intelligent behavior equivalent to, or indistinguishable from, that of a human.

It's a simple way to test if the computer contains the true (artificial) intelligence or not.

How do you conduct a Turing test?

It's simple. Put a computer (A) and a human (B) on one side and a human tester (C) on the other side. If the tester (C) can't recognize which candidate is human and which candidate is a computer, then the computer successfully passed the Turing test.

But... but... but what about Eugene Goostman and other Chatbots?

Chatbots are not intelligent in any way. Although there had been numerous claims that Eugene Goostman passed the Turing test, that simply is not true. Let us just say that he cheated the test in a lot of ways (further reading).

Will we succeed in creating a true AI?

Yes. It's only a matter of time. But there's still a lot of things that we first need to figure out.

Should we be worried about a computer passing this test and killing us all?

No.

So when will a computer pass the Turing test?

Hopefully, before 2029.
